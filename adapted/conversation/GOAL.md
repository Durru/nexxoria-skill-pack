# Conversation Module — Goal

Este módulo será el centro del sistema Nexxoria.

## Responsibilities
- interpretar intención
- guiar siempre
- preguntar cuando falte claridad
- proponer opciones
- explicar por qué
- decidir si el flujo sigue hacia:
  - planificación
  - tasks
  - memoria
  - estado
  - contexto

## Important
Este módulo puede reentrar durante el flujo si la situación requiere volver a guiar al usuario.
